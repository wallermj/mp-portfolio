WEEK 2

WEEK 2

CSS and Github.com

Web Design

In our session today we used Atom and ‘CSS’ to change the text colour. We used GitHub.com to determine where the code of what we wanted to change would be. I then chose a colour I wanted as my title banner would be and then entered it into atom.

By learning how to use ‘CSS’ in regarding colours has enabled me to change the colour of the accents of the page to the exact colour I wanted. By being able to change the colours has also meant that it fits with my colour scheme and overall theme. Atom has enabled me to make the themed layout my own. 

Photography

Today we went over the basics such as Aperture, ISO and Shutter Speed (the exposure triangle) and then we looked at white balance and setting the colours within the image. We looked at the Colour temperatures in the Kevin Scale and how different lights affect what the image looks like. 

The learning of white balance and temperatures of the light has made me aware of when to change the temperature and colours when taking images. This is so that the image isn’t too blue or too orange. White balance has meant it completely balances all the colours out so I don’t have to do it manually each time.

Changing Colours

Web Design

We carried on changing colours of our website making it all similar colours staying on theme. We also had a go at changing the sizes of some of the fonts and the weight as well. Furthermore, we added in some code that changed the layout from the index page to an actual information page with two columns.

by adding different code to atom that changes the structure I was able to change the layout from the index layout to two columns of text with some image placements. I changed this on my design log page so it is readable and not just text.

Photography 

We went through the risk assessment for when we went out on location in the City Centre. The ways to keep both yourself, the rest of the group and the public safe. Additionally, we went over how to set the camera ready for Thursdays shoot day.  

Learning the risk assessment is important for when we go out on location as we want to make it as safe as possible.

Code Snippets

Web Design

I made some small changes on my webpage, such as the layout and titles of each linked page. I also deleted sections I don’t want on my page and added different code snippets

By adding code snippets I have created more space for my content that I will upload further into the module. By erasing the bits, I don’t want has made space for these code snippets. It’s also made each one of my pages different which is what I want for my website as each page will have different content in it.

Photography 

We went out on location in Manchester today to practice our camera skills and adapting to the changes of light. These images will to be put onto my webpage when finished.

Going out on location has meant that I was able to practice taking images that I will use throughout my photography page on my website. This practice has enabled me to figure out what I need to practice and what works well.

Unsplash.com

Web Design

In today’s session we learnt how to insert images form unsplash.com to atom in our code snippet. As well as inserting more code snippets on all the pages so that the page looks more like something, I can add content to rather than a home page throughout.

This was really useful as it enabled me to put place holder images in the code snippet, and I understand how to do this so when I come to adding my own images, I know how to do this.
