WEEK 1

WEEK 1

Introduction to Software

Web Design

We were introduced to the sites and software such as GitHub and Atom, we will be using over the module. We were also introduced to terms that we needed to be aware of over the module, for example ‘HTML’, which is the code used to structure a webpage and ‘CSS’, which is the code used to style a webpage. 

By learning and exploring the software GitHub and Atom this has meant I have created a basic prototype webpage with linked pages. By being able to change the ‘HTML’ and the structure of the page has meant I have created a design I want and not just left it the same as the basic theme. The learning of changing ‘CSS’ has enabled me to change the fonts and colours to match the look I was going for, and where I want to present all my work.

Photography

In the first session of photography we were taught the basic terms and uses of the camera. For example, format, framing, focus, exposure/lighting, shutter speed and image quality. There are all the basic terms of photography. We also were given the cameras to get hands on with the equipment, so we knew early on how to work and use them. Learning the basics of the camera and the settings has meant we can create images that appeal and stand out to who is viewing it. Learning the terms has made it easier to understand the uses of the camera and has overall made the learning of what makes a good image an easier process.

Pushing Up

Web Design

We explored the website of GitHub further by creating a webpage and learning how to push up and publish live on our websites. Additionally, we chose the design layout for our website, I used the theme ‘agency’, and used Atom to start making small changes like the name in the left-hand corner.

Pushing our websites live has meant that everything we have done and changed on the site has saved. Having a theme already as the layout made the process easier as we could then make the changes from then on. 

Photography

In our session today we took images on the camera using a small figure to practice using different f-stops and ISO and aperture. We also were introduced to the rule of thirds, filling the frame and different focus points. 

We practiced taking images with different focal points and different shutter speeds this has meant that we know when to use different settings at the right times and we are able to change them when we feel is necessary.  

What Makes A Good Website?

Digital Wednesday

Web Design

The online session was based on what makes a good website. We were shown a slideshow on responsive web design. We were then asked to research for examples of more web pages in terms of good design and responsiveness. The key elements of responsiveness are innovative, aesthetic, understandable, honest, long lasting and thorough down to last detail.

This was helpful as I got a good understanding of what makes a good website and how it is so important to have a good website that is easy to use for the people viewing the website. I also got ideas of how to layout my website so it readable and easy to view, so it is not complicated to view. 

Photography

We were asked to upload any photography work onto teams.

Linking Pages Together

Web Design

In today’s session we learnt how to link all the pages together so that we had a working website. Additionally, we made text changes on atom, for example, the title changes on the index page from what was already on there to ‘portfolio’. We then pushed this up, so it was live on our website.

Linking all the pages meant that when we clicked on links it sent you to a different page which is what we need when creating a website. It meant that my pages ‘Photography’, ‘Video’, ‘Audio’, ‘Contact’ and ‘Design Log’ were individual pages and not all the index. So, I had a working website, with linked pages. This has also meant that I have been able to change the layout according to the page. For example, on the design log I have changed the layout to a two column text with image options. This is so the text is readable and organised. Additionally, we made css changes, to practice using this tool we changed a few titles to ‘Portfolio’ and adding my name in. 

Photography

We had a guest speaker, who is from Salford FC Media and Marketing department. Ryan spoke about how he started off in the media industry and the different roles he did in order to gather contacts and experience whilst lending a hand to others. He also then discussed what he is currently doing at the club and how he uses social media and websites as the key to reaching and gaining fans.

Typography

Web Design

In today’s session we learnt about Typography and the different fonts that we could use for our own websites. We learnt the difference between san-serif and serif fonts, the baseline and cap line, the ascender and the descender and then tracking and kerning. We discussed that we needed to keep the amount of fonts we use to a minimum, for mine I’ve used two, and finally that we need to limit the line length as it is key to readability of your text. The optimal line length for the body text is considered to be 50-60 characters. 

By learning the differences of fonts I was able to choose two fonts, one san-serif and one serif that will run throughout my website. I only chose two as if you use too many then they all want attention and takes away from what your actually saying.

Photography

Today we looked at how different ISO, Aperture, Shutter Speed and f-stop affected the image if used at different levels. We used a canon website to work out how they all effected and impacted on each other to make the image either very good or too bright/dark. 

Learning the different settings for the camera has been the most important thing for getting good images. Adapting the settings in different lights is also a skill that I’ve been able to learn and use when on shoot. All the images I produce on shoot I will be able to use and transfer them onto my website, for my final assignment.
